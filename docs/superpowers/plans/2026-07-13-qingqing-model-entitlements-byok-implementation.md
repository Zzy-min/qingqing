# 轻青模型选择、权益与 BYOK 实施计划

1. 先为权益、模型可用性、Auto 路由、BYOK 优先级、凭据脱敏和 SSRF 防护编写失败测试。
2. 建立 `/api/v1` 领域 schema、服务层和路由；保留旧 gateway 兼容入口。
3. 实现服务端权益计算、供应商中立模型描述、路由预览和 AgentRun 路由快照。
4. 实现加密凭据元数据与自定义模型 CRUD，保存/测试时重复校验端点安全性。
5. 为 React 增加轻青品牌、Auto 模型选择、路由透明信息和默认关闭的高阶模式。
6. 运行后端全测、前端单测/build、依赖审计、diff 检查和浏览器宽窄屏验收。
7. 独立代码与安全评审；修复高优先级问题后再宣布完成。

## 当前实施进度

- 已完成：供应商中立模型描述、Auto/偏好/锁定路由、Free/VIP 服务端权益、高阶模式门控、加密凭据和自定义模型、SQLite 持久化、邮箱验证码与签名 Bearer 会话、预算审批、幂等 Run、阶段 ModelInvocation、额度账本、文本/图片/语音/音乐/视频执行和作品关联。
- 已完成：React 的 Auto、模型安排、路由预览、预算确认、高阶 API 管理，以及 Flutter 共用 Web/Windows/Android 工程。用户重构后的移动首页/作品/个人中心和桌面三栏工作台已接入服务端偏好、任务轮询、审批、模型透明信息和真实 Artifact 字段，并完成 390、1024、1440 宽度回归测试。
- 安全加固：生产默认不挂载会绕过 Agent 权益与账本的旧 `/api/*`；Google Key 不再进入 URL；自定义 Header 使用白名单；执行状态使用原子抢占；VIP 撤销和凭据撤销会暂停未执行步骤；旧浏览器明文 API Key 会被单向清除。
- 2026-07-14 用户端完整性：补齐移动端灵感/作品快捷入口、桌面端工具箱与最近记录导航、意图搜索和通知中心；后端启停生命周期已迁移至 FastAPI lifespan。
- 当前未完成的生产化工作：生产 KMS/密钥轮换服务、PostgreSQL/Redis/对象存储、独立耐久 Worker、网络层出站代理以彻底消除 DNS 重绑定 TOCTOU、Run 创建时并发与余额的跨进程原子事务、真实 SMTP/供应商凭据联调、React 旧照片/音乐/视频页迁移到 AgentRun、远程 Artifact 的受控下载代理，以及 Windows C++ 构建工具链安装。
